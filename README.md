# Luka's Farm website

Static landing-page template for `lukasfarm.ge`.

## Files

- `index.html` — main page content
- `styles.css` — responsive design and layout
- `script.js` — mobile menu and current year
- `CNAME` — custom GitHub Pages domain

## How to use

1. Copy all files into the root of the GitHub repository.
2. Commit and push to GitHub.
3. In GitHub, open the repository settings.
4. Go to **Pages**.
5. Set the source to deploy from the main branch root.
6. Make sure the custom domain is set to:

```txt
lukasfarm.ge
```

7. In Magticom's DNS settings, connect the domain to GitHub Pages.
